1. Word Quest

Languages used: Python with TKinter library for GUI


Project Description:

A word-guessing game using the tkinter library. It selects a random word from a list of programming language names. The user has six attempts to guess the word, with correct guesses shown in place and incorrect ones hidden. A hint revealing the first letter is available. If the user runs out of attempts, the word is revealed, and they can choose to play again. The GUI includes input fields for guesses, and buttons for submitting guesses, getting hints, refreshing the game, and quitting. The game logic and interface are managed through various functions and tkinter widgets, providing an interactive experience.
